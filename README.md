# Flat UI Colors 

### Clone of Flat UI Colors with additional features, built with

- [React](https://reactjs.org/)
- [Material UI](https://material-ui.com/)

## Getting Started

```
git clone https://github.com/mvximenko/flat-ui-colors.git
```

```
npm install
```

```
npm start
```
